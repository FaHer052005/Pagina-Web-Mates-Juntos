# Mates Juntos - Página Web de Venta de Mates

Bienvenido a la página web de **Mates Juntos**, una tienda en línea de mates artesanales premium.

## Estructura del Proyecto

```
Estructura/
├── index.html          # Página principal
├── css/
│   └── style.css      # Estilos CSS
├── js/
│   └── script.js      # Funcionalidad JavaScript
└── README.md          # Este archivo
```

## Características

✨ **Diseño Responsivo** - Se adapta a cualquier dispositivo (desktop, tablet, móvil)

🛒 **Carrito de Compras** - Funcional con agregar/eliminar productos

📦 **Catálogo de Productos** - 6 productos diferentes con precios

💬 **Formulario de Contacto** - Para consultas de clientes

📱 **Menú Hamburguesa** - Navegación mobile friendly

🎨 **Diseño Moderno** - Colores tierra y estilos elegantes

## Cómo Usar

1. **Abre el archivo `index.html`** en tu navegador web
2. **Navega por la página** usando los respectivos enlaces en el menú
3. **Agrega productos** al carrito haciendo click en los botones
4. **Revisa tu carrito** en la sección correspondiente
5. **Completa el formulario** de contacto para consultas

## Personalización

Puedes personalizar fácilmente:

### Cambiar Colores
Edita las variables en `css/style.css`:
```css
:root {
    --primary-color: #8B6F47;       /* Color principal */
    --secondary-color: #D4A574;     /* Color secundario */
    --accent-color: #2D5016;        /* Color de acento */
    /* ... más variables */
}
```

### Agregar Productos
Duplica la estructura de un producto en `index.html` y cambia:
- El nombre del producto
- La descripción
- El precio
- Llama la función `agregarCarrito()` con los nuevos datos

### Contacto
Actualiza la información de contacto en la sección "Contacto":
- Teléfono
- Email
- Ubicación
